# Selection to CSV

スプレッドシートで**選択した範囲だけ**を CSV ファイルとして保存する Chrome 拡張です。
Google スプレッドシート / Excel Online など、コピー時に TSV（タブ区切り）を出力する表全般に対応します。

## 仕組み

Google スプレッドシートはセルを canvas で描画しており、選択範囲を DOM から直接読み取れません。
そのため本拡張は、**ユーザーがコピーした内容（クリップボードの TSV）を読み取って CSV に変換**します。
この方式のため、特定サイトへのアクセス権限は不要で、TSV でコピーされる表なら何にでも使えます。

## 使い方

1. スプレッドシートで範囲を選択
2. `⌘ + C`（Windows は `Ctrl + C`）でコピー
3. 拡張アイコンをクリック →「クリップボードからCSV保存」

- **Excel向けBOM**: チェックを入れると UTF-8 BOM を付与し、Excel で日本語が文字化けしません（既定 ON）。
- 引用符・カンマ・セル内改行を含むデータも RFC 4180 準拠で正しくエスケープします。

## 権限

- `clipboardRead`: コピーした表データの読み取り
- `downloads`: CSV ファイルの保存

ページの中身は読みません（ホスト権限なし）。処理はすべてローカルで完結します。

## 出力

`selection-<日時>.csv` として保存ダイアログが開きます。
